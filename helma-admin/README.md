# 赫尔玛智能后台管理系统

## 项目简介
赫尔玛智能后台管理系统是衣起赚APP的配套管理平台，用于管理工人端、工厂端的全业务流程，实现数据实时同步、权限分级管控、操作可追溯等核心功能。

## 技术栈
- 前端：Vue 3 + Vite + Element Plus
- 状态管理：Pinia
- 路由：Vue Router
- HTTP客户端：Axios
- 构建工具：Vite
- 代码规范：ESLint + Prettier

## 核心原则
1. 数据实时同步：与前端APP实时互通
2. 权限分级管控：RBAC角色权限模型
3. 操作可追溯：完整操作日志
4. 异常预警机制：自动预警和通知
5. 数据导出能力：Excel格式导出
6. 北极星指标置顶：月度有效到岗闭环数

## 开发要求
- 页面加载速度≤2秒，大数据量查询≤5秒
- 支持按日/周/月/年维度筛选数据
- 适配PC端主流分辨率，响应式布局
- 所有数据卡片支持点击跳转至对应明细模块
- 敏感信息脱敏展示
- 所有操作留存完整日志
